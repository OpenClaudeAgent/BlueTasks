# BlueTasks web app (`@bluetasks/app`)

React + Vite UI for the BlueTasks monorepo. Install, run, and ship instructions live in the **[repository root README](../../README.md)** and **[docs/](../../docs/)** (architecture, quality, i18n, etc.).

From the repo root, run the full dev stack (API + this client):

```bash
npm run dev
```

To work on this package only (with the API already running elsewhere):

```bash
npm run dev --workspace @bluetasks/app
```
